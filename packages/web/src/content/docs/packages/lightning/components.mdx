## Configuration

Add the `@lwc-garden/utils` package to the `modules` config in your `lwr.config.json` file:

```json
// lwr.config.json
{
  "lwc": {
    "modules": [
      {
        "npm": "@lwc-garden/utils"
      }
    ]
  },
}
```
